Technologies page

Css

Css allows developers to separate content from design so that HTML
could perform more of the function that it was originally based on - the markup of content, without worry about the design.

HTML

Hypertext markup language or Html is used for designing online websites and also configure them.
It is used to import codes or text that will be seen on the website. you can put pictures,videos and many other things using html.It is used for putting text in the body and header.

Bootstrap

Bootstrap is used to make your website look better than using just html or css.Using html is just the basics of designing a website.
Bootstrap will make your website look cool and more organized than using Html or css.It is a more easier website designer tool than html or css.
